# Parallax User Guide

## Getting Started

### Installation

1. Download installer for your platform
2. Install:
   - **Windows**: Run `.msi` installer
   - **macOS**: Drag to Applications folder
   - **Linux**: Run `.AppImage` file

### First Launch

1. **API Key Setup**:
   - Get free key from [Google AI Studio](https://aistudio.google.com/app/apikey)
   - Paste in Settings → API Key
   - Stored securely in OS keyring

2. **License (Optional)**:
   - Free tier: 10 AI generations/day, 50 dork limit
   - Pro tier: Enter license in Settings → License

## Features

### 1. Dork Builder

Visual builder for Google dorks.

1. Click operator buttons (site:, intitle:, etc.)
2. Edit query in text area
3. "Try in Google" to test
4. "Save" to store in library

### 2. AI Generator

Natural language → Google dork.

**Examples**:
- "Find exposed database backups"
- "Search for admin login pages"
- "Locate PDF documents about cybersecurity"

**Features**:
- Save generated dorks
- Copy to clipboard
- Try search directly
- Conversation history

**Limits**:
- Free: 10 generations/day
- Pro: Unlimited

### 3. Library

Browse and manage saved dorks.

**Views**:
- Grid: Visual cards
- List: Table view

**Features**:
- Search by name/query/tags
- Filter by category
- Sort by date/name
- Bulk delete (custom dorks only)
- Export to JSON
- Import from JSON

**137 Templates Included**:
- Cloud Storage (AWS, Azure, GCP)
- Databases (MySQL, MongoDB, etc.)
- Login Panels
- Configuration Files
- API Keys & Secrets
- And more...

## Keyboard Shortcuts

- `Ctrl/Cmd + B`: Builder
- `Ctrl/Cmd + G`: AI Generator
- `Ctrl/Cmd + L`: Library
- `Ctrl/Cmd + ,`: Settings
- `Ctrl/Cmd + S`: Save dork
- `Ctrl/Cmd + F`: Search (in Library)

## Troubleshooting

### "Invalid API key"
- Check format (starts with `AIzaSy...`)
- Get fresh key from [Google AI Studio](https://aistudio.google.com)

### "AI Generation failed"
- Check API quota
- Verify internet connection
- Wait 60 seconds and retry

### "Cannot save dork"
- Free tier: 50 dork limit (delete old dorks or upgrade)
- Pro tier: Unlimited

### "License activation failed"
- Check license key spelling
- Verify internet connection
- Contact support with license key
